# Leafletjs-svelte 

Svelte component for [LeafletJS](https://leafletjs.com/) with vector drawing support.


## Components:

- [x] Map
 
**UI layers:**
- [x] Marker
- [x] Popup
- [x] Tooltip

**Raster layers:**
- [x] TileLayer
- [x] TileLayer.WMS
- [x] ImageOverlay
- [x] VideoOverlay

**Vector layers:**
- [x] Polyline
- [x] Polygon
- [x] Circle
- [x] CircleMarker
- [ ] Rectangle

**Other layers:**
- [ ] LayerGroup
- [ ] FeatureGroup
- [ ] GeoJSON

**Controls:**
- [ ] Zoom
- [ ] Layers
- [ ] Scale
